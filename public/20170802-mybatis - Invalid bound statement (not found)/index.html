mybatis - Invalid bound statement (not found)

发表于 2017-08-02 | 分类于 java | 阅读次数

运行报错

使用mybatis生成插件，产生的mapper，由于路径不对移动了一下mapper.java文件

所以造成mapper.xml里面的namespace错误，无法映射

所以把namespace改为正确的即可

例如：

mapper.UserMapper

改为

com.hisen.dao.UserMapper
