2.2 报名

关注各个地方无线电协会，例如:
北京：北京业余无线电服务中心 (公众号)

  • 几乎每个月都会组织考试(如果本地区没有，可以考虑跨地域，全国都可以考)；
  • 准点去报名会比较卡，可以晚点，一般都会有名额；

2.3 考试

考试一般是 30 个题目，对 25 个就行。
很多人开考几分钟就好了，毕竟 A 类操作证题目比较简单。

2.4 拿到操作证

我的情况：

  • 2025.6.21 通过考试
  • 2025.7.23 操作证核发
  • 2025.7.25 操作快递到家

3. 设备登记

3.1 购买设备

一般都是先在各个平台被种草，记住设备型号，
然后去工信部的网站查询是否有『核准代码』
工业和信息化部政务服务平台-结果查询

  • 查看全量设备：业务分类 > 行政许可 > 无线电和卫星通信
  • 查看特定设备：搜索框输入对应关键词即可

我的设备：8800 pro，仅展示，非推荐。

全身主体

3.2 查询操作证号

乌龙：由于我的操作证到手之后，编号部分特别模糊，以至于我以为证件号码就是操作证号。
所以我特意提一下『操作证号』查询( 北京设备登记网站上叫”操作证号”，证书上是”编号”)
查询方式：

  1. 进入 CRAC-中国无线电协会业余无线电分会
  2. 滑动到底部，点击『证书查询』，由于这一步是 ip + port 的形式，我就不给怕后续失效。
  3. 输入『姓名+身份证号』
  4. 得到结果

新版的操作证名称：业余无线电台操作技术能力验证证书
证书信息解读：

  • 编号：操作证号 / 证书编号 ( 叫法有点多，应该统一一下 )
  • 证件号码：S+身份证号

3.2 设备登记

主要是准备好

  1. 操作证号
  2. 设备背面
    • 全部背面
    • 标签部分

一般对图片大小有要求，我是怎么处理的？

  1. 像素：美图秀秀像素编辑，解锁比例，800*1240(北京的要求,不一定准)
  2. 体积：通过 Mac 的另存为，调整质量，使得体积缩小至 100kb
