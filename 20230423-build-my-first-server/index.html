搭建我的第一台服务器

Posted on 2023-04-23 Edited on 2025-07-19 In homelab

1. 简单介绍

当时看朋友买的 E5 CPU 到了白菜价
而且之前使用 dell 9020 sff 装的软路由也转给了朋友
就想着整一个性能更强一些的服务器在家跑着，方便自己折腾。

当前跑的服务

  • DDNS 服务
  • 网心云( 500M下行/100M上行，收益应该可以覆盖网费+电费 )
  • homeassistant
  • 青龙系统( JD 签到 )
