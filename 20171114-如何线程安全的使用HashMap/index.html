Posted on 2017-11-14 Edited on 2025-07-19 In java

线程不安全的原因

  1. HashMap底层是一个Entry数组，一旦发生Hash冲突的的时候，HashMap采用拉链法解决碰撞冲突.
  2. put方法也不是同步的
  3. 扩容的方法也不是同步的

参考：https://www.cnblogs.com/qiumingcheng/p/5259892.html

如何线程安全的使用

  1. Hashtable
  2. ConcurrentHashMap
  3. SynchronizedMap
