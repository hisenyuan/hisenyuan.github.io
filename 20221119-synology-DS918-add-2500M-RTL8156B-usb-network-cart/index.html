给群晖 DS918+ 添加 2.5G 无线网卡

Posted on 2022-11-19 Edited on 2025-11-19 In Tech Symbols count in article: 1.9k Reading time ≈ 2 mins.

0. 前言

给群晖 918+ 添加 RTL-8156B 2.5G 网卡，困扰了我两天。
因为按照之后没法在系统控制面板里面找到对应的网卡。
就连 GitHub 上驱动的作者都说不支持 RTL-8156B(详见：GitHub-issue) 的外置网卡。
但是我想那么多人都买了这种网卡，并且成功了，于是周末到处搜，最终找到了办法。

可能这是 DS918+ 之类才会遇到的问题，最终效果：
