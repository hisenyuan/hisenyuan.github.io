多设备优雅地共享单显示器

Posted on 2025-11-30 In Tech Symbols count in article: 691 Reading time ≈ 1 mins.

1. 缘起

由于家里地方有限，没办法把这三个设备分别安排不同的显示器，只能共享一个。
连接方案：

  • Type-C：Mac Studio
  • HDMI 1：Apple TV
  • HDMI 2：Switch 游戏机

这种方案没什么大问题，就是略费劲(显示器：长按按钮 -> 信号 -> 选择 -> 切换)
有没有更好的解决方案？

  • 共享显示器：多个设备连接到 hub，切换由该设备负责。

2. 选择

选择的尽量照顾后续设备升级的可能，参数尽可能面向未来。
看了不少，最后选了一款 8K 切换器：

  • 三进一出，带遥控器
  • HDMI2.1
  • 视频：8K/60Hz，4K/120Hz ( VRR可变刷新率可达 240Hz )
  • 音频：24bit/192KHz采样率(杜比环绕音效)

原有的 HMDI 线太硬了，顺便买了几根超软的线，方便理线。

购买清单

  • 视频 8K 切换器：https://u.jd.com/W6XpCiv 价格：100+
  • 超软 8K 高清线：https://u.jd.com/WgXZw7a 价格：30+

3. 安装效果

安装完成之后，只需要一根长的线连着显示器。
其它用短线连接，放在开放式设备柜子里面。

3.1 切换器和线材

切换器正面切换器反面线材对比(上软下硬)

3.2 信号切换

信号切换延迟大概 8 秒，体验还是差了点。
但是相比通过显示器按键来切换，还是好很多。

信号1：Mac Studio信号2：Apple TV

4. 后记

现在体验是 OK 了，方便不少。
安装过程也比较简单，麻烦的就是理线~

唯一的缺憾是没法分理出音频，然后共享一个音响。
后续看下怎么折腾，目前的想法是在视频切换器后边，
加个单读的音频分离器，然后把音频信号转到音响上。
可能的问题就是信号会有折损。
